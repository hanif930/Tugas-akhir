Visual Studio Code

Visual Studio Code (VS Code) adalah sebagai aplikasi penyunting kode (code editor) untuk menulis, mengedit, dan mengelola kode sumber (source code) dalam pembuatan perangkat lunak, situs web, atau aplikasi.

Android Studio

Android Studio adalah sebagai wadah tunggal terintegrasi (IDE) untuk merancang, mengode, menguji, dan meluncurkan aplikasi Android.

Adobe Illustrator

Adobe Illustrator adalah untuk membuat dan mengedit grafis berbasis vektor yang dapat diperbesar ke ukuran berapa pun tanpa kehilangan ketajaman atau menjadi pecah.

Figma

Figma adalah sebagai perangkat lunak berbasis web (cloud-based) untuk merancang antarmuka pengguna (UI/UX) situs web, aplikasi seluler, serta media digital lainnya secara kolaboratif.

Borland C++

Borland C++ adalah sebagai perangkat lunak lingkungan pengembangan terpadu (IDE) untuk menulis, mengompilasi (compile), memperbaiki kesalahan (debug), dan menjalankan kode program menggunakan bahasa pemrograman C dan C++.

MySql

MySQL adalah sebagai sistem manajemen basis data (Database Management System / DBMS) untuk menyimpan, mengelola, mengubah, dan mengambil data digital secara terstruktur.

Bahasa Pemrograman

HTML

HTML adalah singkatan dari Hypertext Markup Language, yaitu bahasa markah standar yang digunakan untuk membuat dan menyusun struktur kerangka sebuah halaman website.


CSS

CSS adalah singkatan dari Cascading Style Sheets, yaitu bahasa desain yang digunakan untuk mengatur tampilan dan estetika sebuah halaman website.


JavaScript

JavaScript (JS) adalah bahasa pemrograman tingkat tinggi, dinamis, dan bersifat cross-platform yang menjadi fondasi utama dalam pembuatan website interaktif modern.


Python

Python adalah bahasa pemrograman tingkat tinggi yang dirancang agar mudah dibaca, dipahami, dan ditulis oleh manusia.


Java

Java adalah bahasa pemrograman tingkat tinggi, berorientasi objek (OOP), dan bersifat multiplatform [1, 2].


C++

C++ adalah bahasa pemrograman tingkat tinggi yang merupakan pengembangan dari bahasa C, dengan tambahan fitur Pemrograman Berorientasi Objek (OOP).


Flutter

Flutter adalah sebuah framework atau kerangka kerja UI (User Interface) open-source buatan Google yang digunakan untuk membangun aplikasi multiplatform cantik secara native dari satu basis kode (single codebase) [1].


Kotlin

Kotlin adalah bahasa pemrograman modern, berorientasi objek, dan diketik secara statis (statically-typed) yang berjalan di atas Java Virtual Machine (JVM).
